Blinkit Sales Analysis Dashboard

This project analyzes Blinkit grocery sales data using Microsoft Power BI to uncover insights about product performance, outlet sales, and customer purchasing patterns.

The dashboard helps businesses understand sales trends and make better data-driven decisions.

Project Objective

The objective of this project is to analyze Blinkit sales data and create interactive dashboards that show:

Total sales performance

Product category analysis

Outlet sales comparison

Customer purchasing patterns

The project demonstrates how data visualization tools can transform raw data into meaningful insights.

Tools and Technologies Used

Microsoft Power BI – Data visualization and dashboard creation

Microsoft Excel / CSV – Dataset storage

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – Data calculations

Dataset Information

The dataset contains information related to Blinkit grocery products and outlet sales, including:

Item Identifier

Item Type

Item Fat Content

Item Visibility

Outlet Identifier

Outlet Size

Outlet Location Type

Outlet Type

Item Outlet Sales

This dataset helps analyze product performance and outlet-based sales trends.

Dashboard Features

The Power BI dashboard includes the following visualizations:

Total Sales KPI

Average Sales KPI

Number of Items

Sales by Item Type

Sales by Outlet Size

Sales by Outlet Location

Sales by Fat Content

Outlet Establishment Trend

Interactive filters allow users to explore sales patterns across different outlet types and product categories.

Key Insights

Supermarket outlets generate the highest sales compared to other outlet types.

Certain item categories contribute significantly to overall revenue.

Outlet size and location influence product sales performance.

Sales trends vary depending on product fat content and category.
